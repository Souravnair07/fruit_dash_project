## How to run

### Requirements
- Python 3.6 or above
- Windows, Mac, or Linux

---

### Step 1 — Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/fruit_dash.git
cd fruit_dash
```

### Step 2 — Install dependency (Windows only)
```bash
pip install windows-curses
```
Mac and Linux do NOT need this step — curses comes built in.

### Step 3 — Run the game
```bash
python -m asciiracer
```

If that doesn't work, try:
```bash
python3 -m asciiracer
```
